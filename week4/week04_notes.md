# Week 04 — Operators Notes

## 1.1 — Integer Division & Modulo

| Expression        | Expected | Actual   |
| ----------------- | -------- | -------- |
| `10 / 3`        | 3        | 3        |
| `10.0 / 3`      | 3.33     | 3.333333 |
| `(float)10 / 3` | 3        | 3.333333 |
| `10 % 3`        | 1        | 1        |
| `-7 % 3`        | -1       | -1       |
| `7 % -3`        | -1       | 1        |

## 1.2 — Increment/Decrement

เริ่มต้น `x = 5` ก่อนแต่ละบรรทัด

| Statement | แสดงผลลัพธ์ | x หลังจากนั้น |
| --------- | ---------------------- | ------------------------ |
| `x++`   | 5                      | 5                        |
| `++x`   | 6                      | 6                        |

## 1.3 — Precedence Test

| Expression      | ผลลัพธ์ |
| --------------- | -------------- |
| `2 + 3 * 4`   | 14             |
| `(2 + 3) * 4` | 20             |
| `10 - 2 - 3`  | 5              |
| `2*3 + 4*5`   | 26             |
